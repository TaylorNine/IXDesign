Client

YA

Project

Refer-A-Friend

Project Description

YA wanted to create products out of their digital services, and I was selected to be the UX Designer and Front-End Developer for this undertaking. The project consisted of creating standard products for their offerings which include Instant Win, Rebates and Sweepstake programs. It also included the creation of a completely new product not offered before as a service, the Refer-A-Friend program.

The end goal was to have a prototype that would serve a number of purposes. It would need to be able to clearly demonstrate how this type of program works to a potential client. It needed to serve as a prototype both when our sales team was walking a client through it as well as when they were on their own. Lastly, it needed to satisfy all the requirements of our digital services and development team. The idea was to use one tool that we could demonstrate, sell and re-package quickly without the need to start from scratch.

A lot of caffeine and hours went into the creation of this product, and I couldn’t be happier with the result. I was able to demonstrate it to several potential clients in pitch meetings. I was happy to report that the clients didn’t need any real instruction on how to use the product - which is a great feeling from a User Experience perspective. This prototype directly converted into sales and paid for itself with the first client adoption (which was almost immediately after completion).

This project is the subject of a future case study.

Client

Anheuser Busch

Project

Montejo Sweepstakes

Project Description

In order to create new interest in Anheuser Busch’s Montejo beer, I was tasked with creating a sweepstakes program for it while at YA. The client provided print materials that were accompanying the promotion. We needed a sweepstakes that we could easily hand off to our development team with a caveat: it needed to be bilingual. Due to the way that fulfillment worked, we could also not have two different domains serve up both languages.

To solve this problem, I created some custom jQUERY code that allowed me to input both languages and allowed the user to switch them without another page load. After some quick in house User Testing I came to the conclusion the easiest way to utilize this was with a simple toggle button. Based on the demographic information at hand, the page defaults to English with a Spanish toggle.

Client

Sysco / Naked Juice

Project

Get Naked

Project Description

Sysco wanted to increase their customer’s purchases of Naked Juice drinks so they challenged them to swap out plain orange juice for Naked Juice and offered to foot the bill. This site served as the platform for Sysco customer’s to obtain their rebate using their Sysco Client ID.

The site offers a digital upload or mail-in rebate option that allows customers to either upload their invoice directly or mail it in. When completed they are provided with a confirmation/tracking number for their rebate.
